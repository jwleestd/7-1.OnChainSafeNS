# MVP Rule 파일 정비 Walkthrough

## 변경 요약

PRD v0.5 / SRS v0.8에서 정의한 **Phase-0 Actionable MVP** 기술 스택에 맞추어 프로젝트의 AI Rule 파일을 전면 정비했습니다.

### Before → After

| 항목 | Before (이전 프로젝트) | After (MVP) |
|---|---|---|
| Frontend | Vite + React + TypeScript (SPA) | **Next.js 15 App Router** |
| Backend | Spring Boot 4.0 + Java 21 | **Next.js Route Handlers (TypeScript)** |
| Database | MySQL 8.x + JPA | **Supabase PostgreSQL + Prisma** |
| Architecture | Micro-Service Ready | **2계층 풀스택 모노리스** |
| Deploy | Docker | **Vercel (Git Push)** |
| Auth | OAuth2 (Google, KakaoTalk) | **Admin PW + 이메일 인증** |
